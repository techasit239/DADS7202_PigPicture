ลำดับการทำ
  1. โหลด Thai 150 รูป + parse CVAT classification tags (majority vote)
  2. Apply best segmenter จาก Phase 2 → ได้ masked images ครบ 150 รูป
  3. Setup 5-fold StratifiedGroupKFold (group=piece_id, stratify=class)
  4. วน loop 5 folds รัน 3 classifiers:
      • Exp 1: EfficientNet-B0 (จาก Phase 1)
      • Exp 2: Swin-T (fine-tune ImageNet pretrained)
      • Exp 3: DINOv3 frozen + train cls head
  5. คำนวณ metrics ต่อ fold: Accuracy, Macro-F1, per-class F1, Confusion Matrix
  6. Report mean ± std across 5 folds
  7. Compare 3 classifiers → identify best architecture
Why StratifiedGroupKFold (ไม่ใช่ GroupShuffleSplit): Thai 150 รูปเล็กมาก — single 80/20 split = val 30 รูป variance สูง 5-fold CV รับประกันทุกรูปเป็น val 1 ครั้ง + balance class ทุก fold
Compute time estimate: 5 folds × 3 classifiers × ~20-30 min/train = ~5-7 ชั่วโมง GPU รวม — ต้องวางแผน Colab session ดีๆ
